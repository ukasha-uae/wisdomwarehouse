This is the official communication bridge for **Wisdom Warehouse** in Dubai, UAE.

## About Wisdom Warehouse
Wisdom Warehouse is a creative workspace for kids to explore their curiosity and express their imagination. Located in Al Quoz, Dubai, we offer workshops and artistic discoveries for young minds.

## How to push to GitHub
To push this project to your repository, run the following commands in your terminal:

1. **Initialize Git** (if not already done):
   ```bash
   git init
   ```

2. **Add the remote repository**:
   ```bash
   git remote add origin https://github.com/ukasha-uae/wisdomwarehouse
   ```

3. **Stage and commit your changes**:
   ```bash
   git add .
   git commit -m "Initialize Wisdom Warehouse Portal"
   ```

4. **Push to the main branch**:
   ```bash
   git branch -M main
   git push -u origin main
   ```
