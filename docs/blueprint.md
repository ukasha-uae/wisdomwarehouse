# **App Name**: DailyGlimpse

## Core Features:

- Secure User Authentication: Allows both parents and teachers to securely log in, differentiating access and features based on their role.
- Teacher Daily Activity Posts: Teachers can upload images and videos of student activities, add descriptive captions, and tag specific students for relevant updates.
- AI-Powered Post Descriptions: An AI tool suggests creative and engaging descriptions for daily activity posts based on uploaded media or teacher-provided keywords, saving teachers time.
- Personalized Parent Activity Feed: Parents view a dedicated, personalized feed showing only the daily activities, images, and videos relevant to their child(ren).
- Student Profile Management: Teachers can manage student profiles, assigning them to classes and facilitating content tagging.
- Parent-Child Linking: Allows parents to securely link to their child's profile to access personalized activity updates.

## Style Guidelines:

- Primary color: A warm, inviting blue (#589AF2). This color signifies trust and clarity, reflecting the app's educational and communication purpose. It ensures good contrast on a light background.
- Background color: A very subtle, desaturated light blue (#E9F4FC). It creates a clean, calm canvas for content, visibly connected to the primary color's hue but extremely gentle to keep focus on activities.
- Accent color: A soft, complementary violet (#857AF7). This analogous color provides a friendly contrast for interactive elements and highlights without overpowering the educational content.
- Headline font: 'Alegreya' (serif). Its elegant, contemporary feel provides a touch of intellectual warmth for headings, suiting an educational setting.
- Body font: 'PT Sans' (humanist sans-serif). Paired with Alegreya, PT Sans offers excellent readability for body text and a modern, personable touch that complements the app's community-focused nature.
- Clean, rounded, and easily recognizable illustrative icons that clearly represent actions (e.g., upload, view, message) and educational themes, maintaining a friendly and accessible aesthetic.
- A spacious, content-first layout utilizing a card-based system for daily activity posts, optimizing for visual media (images/videos) and intuitive navigation for both teachers and parents.
- Subtle and purposeful animations for transitions between screens, content loading, and user interactions, designed to enhance the user experience without distraction.
